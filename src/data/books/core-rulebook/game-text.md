To help keep the rules clear, this book adopts some standard ways of presenting information:

**Game Terms:** All game terms are capitalised, so you can tell events from Events, and know when a test should be simple and when you should make a Simple Test.

**Individual Items:** Some game terms encompass many individual items. For example: the game term Spells includes many individual spells a wizard can cast. Each individual item will be italicised. Example: *a Banking Endeavour*; *a Bleeding Condition*; *the Curse Spell*.

**Tests:** When you are asked to make a Test (see page 149), it will be bolded so you can find it easily during play. Example: make an **Average (+20) Perception Test**.

**Acronyms:** The rulebook avoids them as much as possible, but it's important in roleplaying games (RPGs) like Warhammer Fantasy Roleplay (WFRP — we pronounce it WuffRupp) to explain your acronyms the first time the word is used. The Gamemaster (GM) who runs the game can usually clear up any confusion.
